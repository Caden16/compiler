# Lexical analyzer

## Requirements:

> * Python 3+ or Python 2.7
> * Windows  or Linux (macOS untested)
> * tkinter should be installed

## 测试用例

> * 冒泡排序.c
> * 杨辉三角.c

## 结果演示

> * 冒泡排序结果.png
> * 杨辉三角结果.png
> * 演示视频.wmv

## 结果再现步骤

1.在当前目录下打开命令行
2. 输入指令 **python lex.py**
3. 在弹出界面中，点击**File-> Open**
4. 点击**lex**进行词法分词
